# Realtime Chat Application
This realtime chat application is a group project assignment for [network programming lecture](https://ebs.sakarya.edu.tr/Ders/Detay/531786).
This project is being developed using [Node.js](https://nodejs.org/en/docs/), [Socket.IO](https://socket.io/docs/v4), [MongoDB Atlas](https://docs.mongodb.com/cloud/).
